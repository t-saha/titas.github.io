Tech Stack - Python, HTML, Streamlit

Rain Prediction Web Application â€“ Project Overview

This project involves building a machine learning model to predict rainfall using a dataset sourced from a Kaggle competition. The workflow consisted of the following steps:

1. Model Development

A Python script was developed to preprocess the dataset, train a predictive model, and evaluate its performance. The model was saved using the joblib library for later use.

2. Web Application Interface:

A lightweight web interface was created using the Streamlit library to make the model accessible for real-time predictions. This script loads the trained model and allows users to input weather-related features to receive rainfall predictions.

3. Deployment:

The Streamlit app was deployed on the cloud using Streamlit Community Cloud, making it publicly accessible at predict-rain.streamlit.app.

4. Version Control and Workflow:

All development was carried out using Visual Studio Code (VS Code), with version control managed via Git. GitHub was used for repository hosting, and all changes were pushed using Git command-line tools to ensure a structured and collaborative development workflow.
